What is this?

This is a solution to monitor the status of multipe projects, initially it was tailor-made for the needs from a group of specific end-user, it include:

Excel-based data input interface

Flat database (CSV files)

Excel-based project status report



Background:

  • There is a group of merchandisers, buyers, QAs, packaging specialists and managers who work in a retailer business, every year they develop over 700 new products and put them on shelf in stores.

  • Normally people would develop multiple products at the same time, make them to be a product assortment for a sales promotion or everyday range, developing a new product/assortment is project-based.

  • Every new product need to be passed through the "Critical Path" before on shelf. The Critical Path is a set of required tasks such as sample approval, packaging design, purchasing order issusing, pre-shipment inspection...Every task has a completion deadline which depends on specific leadtime and product on-show date (in-store date)

  • Everyone in that retailer would work on multipe projects in parallel and every project would involve multipe persons, thus they need a aligned solution acrossing the teams to monitor the project status
